Detailed Description

Class that represents a cell in the line sweep alghoritm

it is composed by an opening ray(vertical line), a closing ray, two other segments, it remembers the circumstances in wich the cell was open and close (open_case and close_case) and also the last poits of the previous cell and how many points this cell will create in the graph
