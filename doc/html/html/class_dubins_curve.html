double DubinsCurve::addPoints ( double  overlength,
Path &  path 
) const

Method that adds sampled points of this dubins curve to the input path

overlength : length of the final part of the previous curve which has not been inserted in the vector. The first point added will be added at curvilinear length = sampling_length - overlength (if overlength is 0, then the first point will be at 0) in order to preserve the distance between points in the path   path : the path to which the points are added

Returns
: the overlength of this curve
