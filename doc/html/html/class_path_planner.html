)

Method that is recursive; it takea as an input a list of points and two index, and try to go from the starting point to the end point with a dubins curve without collisions. If this succedes the curve is added to the solution, otherwise this mehtod will call itself two times, splitting the path. if there are no more points to split the function it will return false

start_index : the index in the vector of points of the starting point start_angle : the start angle of the starting point end_index : the index in the vector of points of the final point end_angle : pointer to the end angle, if NULL will be chosen randomly graph : the vector of points where to find start and end point solution : the vector to append the correct DubinsCurve

Returns
: true if it was possible to find a solution, false otherwise
