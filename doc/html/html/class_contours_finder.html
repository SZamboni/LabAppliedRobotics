Detailed Description

template<typename T>
class ContoursFinder< T >

Class used to collect the parameters (from an input Json file) of a contour-finding operation and to perform it in the proper way based on the shapes (as templates) we are looking for Invariants: - Maximum 4 elements in the vector params
